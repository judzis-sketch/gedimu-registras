# **App Name**: Gedimų Registras

## Core Features:

- Gedimo registracija: Gyventojai gali užregistruoti gedimą elektroniniu paštu, siunčiant užklausą į sistemą.
- Užklausų nukreipimas: Sistemos valdytojas gauna užklausas ir nukreipia jas atsakingam darbuotojui.
- Darbo pažymėjimas: Darbuotojas atlikęs darbus sistemoje pažymi apie tai.

## Style Guidelines:

- Primary color: Soft blue (#90CAF9) to convey trust and reliability.
- Background color: Light gray (#F5F5F5) for a clean and neutral backdrop.
- Accent color: Teal (#4DB6AC) to highlight important actions and notifications.
- Body and headline font: 'PT Sans', a humanist sans-serif, combines a modern look and a little warmth or personality, and it is suitable for both headlines and body text
- Use clear, descriptive icons for different types of gedimai (e.g., electricity, plumbing) and status indicators.
- Prioritize a simple, intuitive layout for easy navigation and clear task flows.
- Use subtle animations and transitions to provide feedback on user actions (e.g., form submission).
